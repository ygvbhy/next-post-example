## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Database server run

```bash
./pocketbase serve
```

# Next.js 13

- 간단한 Posts 예제를 구현해보면서 Next.js 13의 기본 개념들을 익힌다.
- 데이터베이스는 pocketbase 를 사용한다.
- 데이터베이스는 로컬에서 실행되는 데이터베이스를 사용한다.
- 데이터베이스는 조회, 생성만 가능한 간단한 예제입니다.
